## البدء السريع

### 1. تحميل المشروع وتثبيت المكتبات

```bash
git clone https://github.com/your-org/scamguard-bot.git
cd scamguard-bot
npm install
```

### 2. الإعداد

```bash
cp .env.example .env
# افتح ملف .env وعدّل القيم
```

الحد الأدنى المطلوب:

```
DISCORD_TOKEN=توكن_البوت_هنا
DETECTION_MODE=local          # أو "ai"
```

### 3. إضافة صور الاحتيال

ضع أي صور احتيال معروفة داخل مجلد `scam_images/`. البوت يراقب هذا المجلد أثناء التشغيل — أي صورة تضيفها أو تحذفها تُطبَّق فوراً بدون إعادة تشغيل.

```
scam_images/
  crypto_doubler.png
  fake_nitro.jpg
  phishing_qr.webp
```

### 4. التشغيل

```bash
npm start          # للإنتاج
npm run dev        # للتطوير (يُعيد التشغيل تلقائياً عند تغيير الملفات)
```

## أوضاع الكشف

### الوضع المحلي (الافتراضي)

اضبط `DETECTION_MODE=local`. لا يحتاج اتصالاً بالإنترنت ولا مفتاح API.

**كشف النصوص** — 11 قاعدة بأوزان تغطي:
- كلمات مفتاحية للكريبتو والاستثمار
- وعود بأرباح مضمونة
- روابط مشبوهة
- طلبات التواصل الخاص
- إغراء الهدايا والنيترو المجاني
- سرقة بيانات الاعتماد
- علامات أسلوبية (كل الحروف كبيرة، إيموجي مفرط)

كل قاعدة لها وزن. النتيجة النهائية تُطبَّع بين [0, 1]. الرسائل التي تتجاوز `TEXT_SCORE_THRESHOLD` (الافتراضي `0.6`) تُحذف.

**كشف الصور** — التشابه البصري (pHash):
1. جميع الصور في `scam_images/` تُحوَّل إلى هاش عند بدء التشغيل.
2. المرفقات الواردة تُحوَّل بنفس الطريقة.
3. مسافة هامينج بين الهاشات تحدد مدى التشابه البصري.
4. الصور التي تتجاوز `IMAGE_SIMILARITY_THRESHOLD` (الافتراضي `0.85`) تُحذف.

### وضع الذكاء الاصطناعي

اضبط `DETECTION_MODE=ai` وأضف `AI_API_KEY`.

المزودون المدعومون:

| المزود | اضبط `AI_PROVIDER=` | النموذج الموصى به |
|---|---|---|
| OpenAI | `openai` | `gpt-4o-mini` |
| Anthropic | `anthropic` | `claude-haiku-4-5-20251001` |

النموذج يُعيد حكماً بصيغة JSON منظمة (`isScam`, `confidence`, `category`, `reason`). الرسائل والصور التي تتجاوز `AI_THRESHOLD` (الافتراضي `0.75`) تُحذف.

## مرجع الإعدادات

| المتغير | الافتراضي | الوصف |
|---|---|---|
| `DISCORD_TOKEN` | **مطلوب** | توكن البوت |
| `DETECTION_MODE` | `local` | `local` أو `ai` |
| `IGNORED_CHANNELS` | `` | معرّفات القنوات المستثناة (مفصولة بفاصلة) |
| `IMMUNE_ROLES` | `` | معرّفات الرتب المحصّنة من الحذف (مفصولة بفاصلة) |
| `AI_PROVIDER` | `openai` | `openai` أو `anthropic` |
| `AI_API_KEY` | `` | مفتاح API (لوضع الذكاء الاصطناعي فقط) |
| `AI_MODEL` | `gpt-4o-mini` | اسم النموذج |
| `AI_THRESHOLD` | `0.75` | حد الثقة لوضع الذكاء الاصطناعي |
| `IMAGE_SIMILARITY_THRESHOLD` | `0.85` | حد تشابه الصور |
| `TEXT_SCORE_THRESHOLD` | `0.60` | حد نقاط النصوص |
| `ACTION_DELETE` | `true` | حذف رسائل الاحتيال |
| `ACTION_SEND_WARNING` | `true` | إرسال تحذير في القناة |
| `ACTION_TIMEOUT_MINUTES` | `0` | كتم العضو لمدة N دقيقة (0 = معطّل) |
| `ACTION_LOG` | `true` | كتابة سجلات منظمة |

## إضافة قاعدة نصية جديدة (الوضع المحلي)

افتح `src/detectors/localTextDetector.js` وأضف كائناً في `TEXT_RULES`:

```js
{
  name:   "اسم_القاعدة",
  weight: 0.25,                    // من 0 إلى 1، مساهمة في النقاط الإجمالية
  test:   (t) => /نمطك_هنا/i.test(t),
},
```

لا يحتاج أي تغيير في بقية الكود.

---

## إضافة كاشف جديد

1. أنشئ `src/detectors/myDetector.js` يرث من `BaseDetector`.
2. طبّق `async init()` و `async analyze(payload)`.
3. أعد النتيجة باستخدام `this._result(isScam, confidence, reason, meta)`.
4. اربطه في `MessageHandler` أو `ImageHandler` أو كليهما.

## صلاحيات البوت في ديسكورد

البوت يحتاج الصلاحيات التالية في كل سيرفر:

- **قراءة الرسائل / مشاهدة القنوات**
- **إرسال الرسائل** (لإرسال تحذيرات الاحتيال)
- **إدارة الرسائل** (لحذف رسائل الاحتيال)
- **كتم الأعضاء** (اختياري — فقط إذا كان `ACTION_TIMEOUT_MINUTES > 0`)
- **صلاحية محتوى الرسائل** (فعّلها في بوابة المطورين)
